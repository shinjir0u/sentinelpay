# Sentinel App

The app is designed to practice the core concepts of spring boot learned from the Telusko Java/Spring course. (https://www.udemy.com/course/spring-5-with-spring-boot-2/).
The core concepts include spring boot with maven, spring security, spring ai, spring data jpa, spring aop and docker.
The app is structured as a multi-module project with each feature as a module for scalability.

## Phase 1

Implement the transfer logic between accounts.
Basic CRUD are skipped as it's redundant for the main purpose of the project is learning.
